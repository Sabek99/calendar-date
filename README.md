# calendar-date
A web app that references to a particular day represented within some calendar systems


## Requirements
 `pip install -r requirements.txt`
 
## Run
 ` python main.py`
 
## Demo
 deployed to [Heroku](https://calendar-date.herokuapp.com/?lang=ara,en)
